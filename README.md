## Setup
#### Requirements:
- [NodeJS](http://nodejs.org/) >= LTS (18.16.0) (linux: [nvm](https://github.com/nvm-sh/nvm))
- [pnpm](https://pnpm.io/installation) >= 8.x (installation via corepack is ok)

#### Install deps and start:
1. Clone this repo
2. Install deps
    ```sh
    pnpm install
    ```
3. Start
    ```sh
    pnpn dev
    ```
